# One piece css gallery

A creative front-end project showcasing advanced CSS animations and interactive DOM manipulation. This gallery brings the Straw Hat crew to life with an infinite scrolling banner and a 3D interactive mosaic effect.

## Live Demo
Check out the live project here: [https://pir4teking.github.io/one-piece-css-mosaic/](https://pir4teking.github.io/one-piece-css-mosaic/)

## Key Features
- *Infinite CSS Animation:* Seamless background scrolling using custom @keyframes.
- *3D Card Flip:* Interactive hover effects to reveal hidden mosaic pieces.
- *Dynamic State Management:* JavaScript-based click logic to lock cards in position and complete the mosaic puzzle.
- *Responsive Grid:* Built using CSS Grid for perfect alignment across different screen sizes.

## Technologies Used
- HTML5
- CSS3 (Animations & Transforms)
- Vanilla JavaScript

## How to explore
1. Open the link above.
2. Hover over the grid cards to trigger the flip animation.
3. Click on the cards to lock them in place and reveal the full pirate poster.
